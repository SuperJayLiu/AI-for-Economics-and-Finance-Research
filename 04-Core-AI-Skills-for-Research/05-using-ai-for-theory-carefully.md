# Using AI For Theory Carefully

AI can help explain assumptions, intuition, and proof structure. It can also make algebraic and logical errors.

Useful tasks:

- clarify notation
- restate assumptions
- explain intuition
- identify missing cases
- rewrite proposition statements
- generate proof-check questions

Risky tasks:

- unverified derivations
- equilibrium existence claims
- comparative statics
- welfare conclusions
- claims about uniqueness or stability

Every derivation must be checked manually.
