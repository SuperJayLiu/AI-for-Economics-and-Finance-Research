# 面向经济学和金融学研究的 AI 手册

这是一个 GitHub 原生的手册和资源库，帮助经济学和金融学研究者负责任地使用 AI 和大语言模型。

本仓库不是“提示词合集”，也不是“AI 工具清单”。

它关注的是如何建立可靠的 AI 辅助研究工作流：什么时候可以用 AI，什么时候不应该用，如何核查输出，如何保护数据，如何组织研究项目，以及如何跟上快速变化的 AI 工具。

## 适合谁阅读

- 博士生和研究助理
- 青年教师和应用经济学研究者
- 金融学研究者和政策研究者
- 希望设计 AI 研究工作坊的教师
- 需要统一 AI 使用规范的研究团队

## 核心原则

AI 可以自动化劳动，但不能替代学术责任。

对于严肃研究，建议使用：一个 GitHub 仓库管理研究项目，一个 ChatGPT 或 Claude Project 管理 AI 工作空间，一个 AI 使用日志记录可追溯性，一个清晰的文件夹结构管理代码、数据、论文和结果。
