# Context Windows And Files

AI systems can only use information available in their current context, tool outputs, memory, or connected knowledge sources. Uploading a file does not guarantee the model read every page correctly.

For long papers, codebases, or replication packages:

- split tasks into smaller units
- ask the AI to state which files or sections it used
- require page, table, equation, or line references where possible
- verify important claims manually

AI may miss appendices, footnotes, table notes, data dictionaries, and code comments. These details often matter in economics and finance.
