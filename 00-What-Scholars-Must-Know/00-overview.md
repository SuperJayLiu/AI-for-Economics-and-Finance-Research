# What Scholars Must Know

This section explains the minimum concepts economics and finance scholars need before using AI in research.

Key points:

- LLMs are not databases.
- AI can generate fluent falsehoods.
- Search, reasoning, writing, and coding are different tasks.
- File upload does not guarantee source-grounded answers.
- RAG and agents can help, but they create new verification duties.
- Local models may reduce privacy risk, but they do not remove accuracy risk.
