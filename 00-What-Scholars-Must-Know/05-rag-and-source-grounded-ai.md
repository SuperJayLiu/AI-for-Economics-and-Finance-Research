# RAG And Source-Grounded AI

Retrieval-augmented generation, or RAG, connects an AI model to a source collection such as papers, notes, documentation, or a literature library.

RAG can help when:

- the source collection is curated
- retrieved passages are visible
- answers cite specific source locations
- the user checks whether the cited passages support the answer

RAG does not guarantee correctness. The model can retrieve irrelevant text, ignore key passages, or overstate what a source says.

For research, source-grounded AI is useful only when source checking is part of the workflow.
