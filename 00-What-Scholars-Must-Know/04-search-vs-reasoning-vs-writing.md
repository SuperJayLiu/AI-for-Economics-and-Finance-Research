# Search Vs Reasoning Vs Writing

Do not use one AI mode for every task.

Search-oriented tasks:

- find papers
- check current policies
- identify tool documentation
- verify public facts

Reasoning-oriented tasks:

- compare mechanisms
- critique identification
- explain code logic
- diagnose structure in an argument

Writing-oriented tasks:

- improve clarity
- reorganize paragraphs
- produce seminar summaries
- draft response language

Each task has a different verification burden. Search needs source checking. Reasoning needs expert review. Writing needs meaning preservation.
