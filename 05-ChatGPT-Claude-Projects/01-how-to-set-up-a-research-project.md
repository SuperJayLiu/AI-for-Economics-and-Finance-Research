# How To Set Up A Research Project

Use a separate AI project for each research job.

## Setup Steps

1. Define the research task.
2. Decide what materials are safe to upload.
3. Write project instructions.
4. Specify output formats.
5. Specify what AI must flag as uncertain.
6. Add verification rules.
7. Keep a research log outside the AI tool.

## Basic Project Instructions

Use instructions such as:

- Prioritize accuracy over fluency.
- Never invent citations, data, coefficients, or institutional details.
- Separate facts from interpretation.
- Flag uncertain claims.
- Identify what must be checked manually.
- Preserve economics and finance terminology carefully.
- Do not make causal claims stronger than the evidence supports.
