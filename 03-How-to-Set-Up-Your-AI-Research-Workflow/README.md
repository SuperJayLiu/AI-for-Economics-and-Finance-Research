# Set Up AI Research Workflows

This folder contains complete AI project setups for recurring economics and finance research jobs.

## What This Folder Is For

Use it when you want to create a reusable AI workspace, such as:

- literature review project
- empirical paper project
- theory paper project
- replication project
- referee report project
- seminar presentation project
- revision response project
- conference tracker project

## How It Differs From Tools and Instructions

The instruction folder gives reusable pieces. This folder combines those pieces into complete research workflows with purpose, safe inputs, project instructions, outputs, verification, and common mistakes.
