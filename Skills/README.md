# Skills

Copy-paste AI skills for economics and finance research.

## Paper Spine

Use when a draft, outline, or paper idea feels unfocused.

Prompt:
Act as an economics and finance paper-spine assistant. Identify the central research question, strongest supportable contribution, argument spine, claim-evidence map, revision matrix, revised outline, and verification checklist. Do not invent citations, mechanisms, data, coefficients, or robustness checks. Do not make the contribution stronger than the evidence.

## Claim Support Bank

Prompt:
Split my manuscript section into citable claim units. For each claim, classify needed support as background, theory, institution, data source, empirical precedent, method, or limitation. Use only papers or sources I approve. Return claim, support needed, source, support strength, reason, and manual check note.

## Evidence-Aware Rewrite

Prompt:
Rewrite this section while preserving evidence boundaries. First identify each paragraph claim and mark it established, suggestive, speculative, or unsupported. Then rewrite for clarity, keep hedging aligned with evidence, and flag overclaiming.

## Table And Figure Audit

Prompt:
Audit this economics or finance table or figure. Check question answered, sample, period, units, variables, estimation choices, uncertainty, caption, reproducibility, and likely referee questions. Return revised note, audit checklist, improvements, and verification needed.

## Data Lineage

Prompt:
Create a DATA.md draft from the data sources, access rules, raw file list, processing code, and outputs I provide. Include restrictions, processing steps, sample construction, rebuild steps, and sharing limits. Do not infer permissions.

## Paper To Slides

Prompt:
Turn this economics or finance paper into slides around the argument spine, not manuscript section order. For each slide give title, bullets, evidence role, speaker note, and likely question. Add limitations and backup slides.

## Referee Response Matrix

Prompt:
Turn editor and referee reports into a response matrix with columns for comment, theme, severity, required action, evidence or code needed, draft response, and status.

## Code Review

Prompt:
Review this economics or finance code. Explain blocks, inputs, outputs, sample construction, merges, filters, timing, units, fixed effects, clustering, weights, standard errors, and diagnostics to run before accepting changes.
