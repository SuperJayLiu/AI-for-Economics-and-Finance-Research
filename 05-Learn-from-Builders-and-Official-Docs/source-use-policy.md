# Source Use Policy

This guide draws on public work by economists, finance scholars, AI workflow builders, and official tool documentation.

It does not simply collect links. It extracts reusable workflow patterns, cites sources briefly where relevant, and adapts the ideas into economics and finance research instructions.

## Rules

- cite briefly in relevant pages
- avoid long biographies
- do not copy external skills wholesale
- link to original sources
- record last-checked dates for tool behavior
- separate official documentation from practitioner judgment
