# Learn From Builders and Official Docs

This folder explains what the handbook draws from external builders, scholars, public resources, and official documentation.

## What This Folder Is For

Use it to see:

- which external sources influenced the repo
- what workflow ideas were adapted
- where official technical behavior should be checked
- how to avoid turning resources into an uncurated link dump

## How It Differs From Tools and Instructions

This folder is about source context and learning strategy. It is not where directly reusable instructions live.

## Source-Use Policy

This guide draws on public work by economists, finance scholars, AI workflow builders, and official tool documentation. It does not simply collect links. It extracts reusable workflow patterns, cites sources briefly where relevant, and adapts the ideas into economics and finance research instructions.

Tool behavior changes quickly, so tool-specific notes should be dated and periodically reviewed.
