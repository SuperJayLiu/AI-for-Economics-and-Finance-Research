# Empirical Research And Coding

This section focuses on practical coding and empirical workflows in economics and finance.

AI can help with:

- Stata, R, Python, MATLAB, and LaTeX
- variable construction
- regression tables
- event studies
- panel data
- text as data
- debugging
- code review
- reproducibility

AI-generated code should never be trusted without running it, inspecting outputs, and testing small examples.
