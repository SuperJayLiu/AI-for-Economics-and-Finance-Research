# Minimum Setup For Scholars

## What Problem This Solves

Beginners often ask whether they need every AI tool at once. They do not. The goal is a small, safe setup that supports real research workflows.

## Recommended Minimum Setup

1. ChatGPT or Claude for general research assistance.
2. GitHub account for version control and recovery.
3. VS Code or another editor for manuscripts, code, and project files.
4. Zotero or another reference manager.
5. One research language: Stata, R, or Python.
6. A private folder structure for each research project.
7. Basic Git usage: commit, branch, diff, restore.

## What To Learn First

- how to give context to AI
- how to ask for uncertainty and verification checks
- how to keep raw data separate from edited files
- how to inspect AI changes before accepting them
- how to log AI-assisted work

## What To Avoid At The Beginning

- using agents on important files without Git
- uploading restricted data
- asking AI to decide identification strategy
- using tool comparisons that are not dated
- collecting prompts without understanding workflow

## Safest First Workflow

Start with a literature note workflow using papers you are allowed to process. Ask AI to structure notes, identify claims to verify, and separate fact from interpretation.

## Sources and Workflow Influences

Draws on public academic AI workflow guides and official documentation for projects, skills, GitHub, and coding agents.
