# Tools And Features

This section explains AI feature types without turning the handbook into a tool directory.

The goal is to understand what features do and how to evaluate them:

- chatbots
- search tools
- coding tools
- file upload
- web search
- code execution
- projects and knowledge bases
- agents
- memory and personalization
- GitHub and version-control integration

For every feature, ask whether it improves a real research workflow, reduces errors, improves reproducibility, or protects time.
