# Prompts Projects Skills Agents MCPs And GitHub Repos

## What Problem This Solves

Scholars often confuse one-time prompts, persistent projects, reusable skills, agents, connectors, and GitHub repositories.

| Concept | Meaning for scholars |
| --- | --- |
| Prompt | One-time instruction |
| Project | Persistent workspace for a paper or task |
| Skill | Reusable procedure for a repeated task |
| Agent | AI system that plans and uses tools |
| MCP | Connector between AI and external apps or data |
| GitHub repo | Version-controlled research workspace |
| `AGENTS.md` / `CLAUDE.md` | Instructions for AI working inside the repo |

## Practical Rule

Use prompts for one-off tasks, projects for ongoing context, skills for repeated workflows, agents for tool-using execution, MCPs for connections, and GitHub for traceability.

Draws on official OpenAI, Anthropic, MCP, AGENTS.md, GitHub, and Git documentation.
