# What AI Skills Do Econ Finance Scholars Need?

Economics and finance scholars need AI skills that support research judgment.

High-value skills:

- giving task-specific context
- decomposing research work into steps
- distinguishing facts from interpretation
- verifying citations and claims
- using AI for code review and debugging
- building project workspaces
- keeping research logs
- protecting confidential data
- using version control
- evaluating new tools without chasing hype

Low-value habits:

- collecting prompt lists without workflow design
- using AI output as literature knowledge
- asking broad questions with no verification plan
- switching tools constantly
