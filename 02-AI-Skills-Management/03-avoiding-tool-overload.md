# Avoiding Tool Overload

Most scholars need a small, stable AI toolkit.

Evaluate tools by asking:

- Does it solve a real research problem?
- Does it reduce or increase verification burden?
- Does it handle files, code, or sources in a way I can inspect?
- Does it fit my privacy obligations?
- Can I reproduce what it did?
- Can I test it on a small example?

Do not adopt a tool because it is new. Adopt it because it improves a workflow you already understand.
