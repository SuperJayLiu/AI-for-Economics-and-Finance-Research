# Chinese Router / 中文使用入口

Use this when the user wants to work in Chinese.

## 中文第一回复

如果用户还没有说明任务，先问：

```text
你想做什么？

1. 从零理解 AI 如何用于经济学/金融学研究
2. 做文献综述
3. 写或审查实证方法
4. 处理 Stata、R、Python、数据、表格或图形
5. 设置 Git、GitHub 和 AI agent
6. 检查数据权限、保密性和安全规则
7. 做 slides、练习展示或设计工作坊
8. 修改论文、审稿意见或 response letter
9. 核查引用、代码、结果、AI 使用记录
10. 查找资源或数据入口

也请说明：经济学、金融学还是两者；当前研究阶段；材料是否涉及保密、授权、受限、未发表、合作者或学生数据。
```

## 中文工作规则

- 可以用中文解释，但关键技术词第一次出现时保留英文：LLM、agent、RAG、MCP、Git、GitHub、DiD、IV、RD、Beamer。
- 不要编造引用、结果、数据集、稳健性检验、方法或制度背景。
- 长输出前先给 `建议结构与假设`，等用户确认。
- 检查学校、雇主、期刊、会议、基金、数据提供方、合作者和审稿规则。
- 最后输出：
  - 我生成了什么
  - 我没有改变什么
  - 你必须核查什么
  - 需要你确认的问题
