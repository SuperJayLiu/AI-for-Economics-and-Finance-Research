# AI Use Log

## What Problem This Solves

An AI-use log connects AI assistance with reproducibility, disclosure, Git history, and academic accountability.

## Template

```text
Date:
Tool/model:
Task:
Files touched:
Output accepted:
Human edits:
Checks run:
Remaining uncertainty:
Commit hash:
Disclosure needed:
```

## Example

```text
Date: 2026-05-24
Tool/model: ChatGPT
Task: revised introduction structure
Files touched: paper/introduction.tex
Output accepted: outline only
Human edits: rewrote all paragraphs manually
Checks run: verified all citations and result claims
Remaining uncertainty: contribution framing needs coauthor review
Commit hash: pending
Disclosure needed: likely no, internal log retained
```

## Rule

If AI touched files, code, data, or paper text, log what was accepted and what was checked.
