# Automated Presentation Generation

## What Problem This Solves

AI can help convert a paper, report, or workshop into a slide deck. Automation is useful only if the presentation remains accurate, audience-aware, and verifiable.

## Two Recommended Output Styles

| Style | Best for | Main risk |
| --- | --- | --- |
| Interactive HTML slides | public talks, teaching, web sharing, explainers | excessive animation or public exposure |
| Traditional LaTeX/Beamer slides | seminars, academic conferences, math-heavy talks | dense slides or brittle compilation |

## Recommended Automation Loop

1. Ask AI for a talk plan.
2. Review the plan.
3. Provide verified figures and tables.
4. Generate a draft deck.
5. Check every claim and visual element.
6. Export to PDF or shareable HTML.
7. Rehearse with AI as an audience.
8. Commit source files and exported artifacts.

## What Not To Automate

- deciding the contribution
- choosing which robustness checks matter
- rewriting causal claims
- inventing figures
- answering hard questions without author review
