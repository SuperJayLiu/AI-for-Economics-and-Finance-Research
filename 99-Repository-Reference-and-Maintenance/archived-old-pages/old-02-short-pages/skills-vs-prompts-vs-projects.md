# Skills Vs Prompts Vs Projects

## What Problem This Solves

Many researchers use AI as if every task were a prompt. Repeated research work needs stronger structures.

| Concept | Best use |
| --- | --- |
| Prompt | One-time request |
| Project | Persistent workspace for a paper or task |
| Skill | Reusable procedure for repeated work |
| Agent | Tool-using system that can plan and act |
| MCP | Connector between AI and external tools or data |
| GitHub repo | Version-controlled research workspace |

## Rule

Prompts are for one-time tasks. Skills are for repeated workflows. Projects are for persistent research contexts. Agents are for multi-step tool-using execution.
