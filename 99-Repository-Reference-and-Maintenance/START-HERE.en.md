# Start Here

This handbook is not a prompt collection. It teaches controlled AI workflows for economics and finance research.

## First Rule

AI output is not evidence. It is a draft, explanation, checklist, critique, or coding aid that must be checked against sources, data, code, or theory.

## Minimum Setup

Start with:

1. ChatGPT or Claude
2. GitHub account
3. VS Code or another editor
4. Zotero or another reference manager
5. One coding language: Stata, R, or Python
6. A private folder structure for research projects
7. Basic Git usage

Read: [Minimum Setup for Scholars](../01-Start-Here-to-Learn-AI-for-Econ-Finance-Research/what-scholars-must-know/00-minimum-setup-for-scholars.md)

## A Safe Beginner Workflow

1. Pick one narrow research task.
2. Give the AI only the context needed for that task.
3. Ask for structured output with uncertainty flags.
4. Verify every factual claim, citation, coefficient, equation, and institutional detail.
5. Record what AI helped with and what you accepted.
6. Keep original files under version control.

## A Good First AI Project

Start with a literature review project, not a full paper-writing project. Literature work makes the benefits and risks visible: AI can organize and compare ideas, but it may also invent citations or misstate contributions.

Use: [Literature Review Project](../03-How-to-Set-Up-Your-AI-Research-Workflow/ai-projects/03-literature-review-project.md)
