# How To Use External Skills Responsibly

## Rules

1. Check the license.
2. Cite the original author.
3. Link to the original source.
4. Do not copy large amounts of text unless the license allows it.
5. Prefer adapting the idea into your own economics or finance workflow.
6. Record the date checked.
7. Test the skill on a small example before using it in real research.
8. Never use an external skill on confidential data before reviewing what it does.

## Evaluation Checklist

- What task does the skill solve?
- What inputs does it require?
- Does it protect data?
- Does it include verification steps?
- Is it current?
- Is it field-appropriate for economics or finance?
