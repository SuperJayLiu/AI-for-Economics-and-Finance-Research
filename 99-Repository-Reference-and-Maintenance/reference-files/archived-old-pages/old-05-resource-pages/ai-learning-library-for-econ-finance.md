# AI Learning Library For Economics And Finance

## What Problem This Solves

Economics and finance scholars do not need an endless AI link list. They need a curated path for what to learn next, which resources are worth attention, and how each resource connects to research work.

## Learning Sequence

Use this sequence as the default learning path:

1. AI fundamentals
2. AI for academic reading
3. AI for coding and data work
4. AI for writing and revision
5. AI Projects and Skills
6. GitHub and reproducibility
7. Claude Code, Codex, Cursor, and other coding agents
8. agents and automation
9. staying updated without hype

## Resource Card Format

Use cards instead of raw links:

```text
Resource name:
Type: YouTube / Newsletter / GitHub / Blog / Official docs / Course
Level: Beginner / Intermediate / Advanced
Best for: Coding / Writing / Agents / Research workflow / Tool updates
Why it matters for economics and finance scholars:
How to use it:
Caution:
Last checked:
```

## Inclusion Rule

Include a resource only if it teaches a reusable workflow, improves research quality, documents tool behavior, gives working examples, explains risks, or helps scholars build durable AI skills.

## Sources And Workflow Influences

Draws on curated AI learning-library approaches, especially the idea of reducing information overload through beginner-friendly resource paths and practical tool/workflow curation.
