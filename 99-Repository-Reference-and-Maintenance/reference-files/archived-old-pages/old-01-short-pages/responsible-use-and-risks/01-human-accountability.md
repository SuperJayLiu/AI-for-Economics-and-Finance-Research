# Human Accountability

AI tools cannot take scholarly responsibility. The researcher is accountable for:

- factual claims
- citations
- data use
- code
- empirical interpretation
- theoretical derivations
- disclosure
- confidentiality

Use AI as assistance, not authority.

Before using AI output in research, ask:

- Did I verify the claim?
- Did I inspect the code or derivation?
- Did I preserve the meaning of the original source?
- Did I follow journal, funder, institution, and data-use rules?
- Should AI assistance be disclosed?
