# SSRN Working Paper Manuscript

This folder contains a standalone English LaTeX manuscript version of the repository:

- `ai-for-economics-and-finance-research-handbook.tex`

The manuscript is designed as a citable working paper for SSRN or similar working-paper platforms. It starts with the handbook, then turns the repo's skills, templates, agent workflows, examples, datasets, links, and teaching materials into a single readable document.

Suggested compile command:

```bash
pdflatex ai-for-economics-and-finance-research-handbook.tex
pdflatex ai-for-economics-and-finance-research-handbook.tex
```

If your LaTeX environment supports `latexmk`, use:

```bash
latexmk -pdf ai-for-economics-and-finance-research-handbook.tex
```
