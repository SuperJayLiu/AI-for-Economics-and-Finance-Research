# Why GitHub Matters When Using AI

AI can create many changes quickly. GitHub helps you inspect, discuss, reverse, and document those changes.

Useful practices:

- commit stable versions
- use branches for experiments
- review diffs before accepting AI edits
- keep README files for project structure
- separate raw data from generated outputs
- log AI-assisted changes

The goal is not to make every scholar a software developer. The goal is to make research changes traceable.
