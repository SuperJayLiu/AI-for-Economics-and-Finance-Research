# GitHub Version Control And Research Safety

GitHub and version control are not only for software engineers. They protect research projects when AI tools can edit files, generate code, or restructure documents.

Version control helps prevent:

- file loss
- silent overwrites
- unwanted AI changes
- confusion between raw and cleaned data
- unreproducible code edits
- unclear histories of analysis choices

Core practice: before using AI to change files, make sure you can see and undo the changes.
