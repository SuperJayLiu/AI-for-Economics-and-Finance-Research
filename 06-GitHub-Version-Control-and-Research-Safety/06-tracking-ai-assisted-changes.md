# Tracking AI-Assisted Changes

Track AI-assisted work so you can reproduce, disclose, and audit your research process.

## Minimal Log Template

```text
Date:
Tool/model:
Research task:
Input materials:
AI output used:
Human changes made:
Files changed:
Verification performed:
Remaining uncertainty:
Disclosure needed:
```

## Example

```text
Date: 2026-05-24
Tool/model: ChatGPT
Research task: helped revise introduction structure
Input materials: intro_v2.tex, project abstract
AI output used: paragraph organization only
Human changes made: rewrote all claims and transitions manually
Files changed: intro_v3.tex
Verification performed: checked all citations and empirical claims manually
Remaining uncertainty: need to verify literature positioning
Disclosure needed: likely no, but record kept for transparency
```

This is a research accountability practice, not a bureaucratic exercise.
