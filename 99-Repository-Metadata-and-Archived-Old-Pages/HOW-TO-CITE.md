# How To Cite This Repository

Use [CITATION.cff](CITATION.cff) for citation metadata.

## Suggested Citation

Liu, Chaojie. 2026. *AI for Economics and Finance Research*. GitHub repository. https://github.com/SuperJayLiu/AI-for-Economics-and-Finance-Research

## BibTeX

```bibtex
@misc{liu2026aiforeconfinance,
  author = {Liu, Chaojie},
  title = {AI for Economics and Finance Research},
  year = {2026},
  howpublished = {GitHub repository},
  url = {https://github.com/SuperJayLiu/AI-for-Economics-and-Finance-Research}
}
```

## Citing A Specific Page

Include the page title, repository title, URL, and access date.

## Citing A Release

When stable releases are available, cite the release version and date.
