# 从这里开始

本手册不是提示词合集。它教你如何为经济学和金融学研究建立可核查、可复现、可追踪的 AI 辅助工作流。

## 第一原则

AI 输出不是证据。它只能作为草稿、解释、清单、批评意见或代码辅助，必须用原始文献、数据、代码或推导来核查。

## 最小设置

建议从这些开始：

1. ChatGPT 或 Claude
2. GitHub 账号
3. VS Code 或其他编辑器
4. Zotero 或其他文献管理器
5. 一门常用研究语言：Stata、R 或 Python
6. 一个私有的研究项目文件夹结构
7. 基础 Git 使用能力

## 安全的第一个工作流

1. 选择一个很小的研究任务。
2. 只给 AI 必要的上下文。
3. 要求结构化输出，并标记不确定之处。
4. 人工核查事实、引用、系数、公式和制度细节。
5. 记录 AI 帮助了什么，你采纳了什么。
6. 用 Git 或其他版本管理方式保存原始文件。
