# Preventing Unwanted AI Changes

AI coding tools and agents can edit more than intended.

Use these safeguards:

- work on a branch
- keep raw data read-only
- review diffs before committing
- ask AI to explain intended edits before applying them
- limit file scope
- run tests or small examples
- keep generated outputs separate from source files

If a tool cannot show what it changed, do not use it on important research files.
