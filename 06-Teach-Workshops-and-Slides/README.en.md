# Teach Workshops and Slides

This folder is for people who want to teach responsible AI use in economics and finance research.

## What This Folder Is For

Use it for:

- workshop outlines
- slide-ready Markdown
- class exercises
- PhD student training
- faculty training
- RA onboarding
- research group norms

## How It Differs From The Handbook

The handbook is for self-study. This folder turns the material into teachable sessions, classroom activities, and lab onboarding materials.
