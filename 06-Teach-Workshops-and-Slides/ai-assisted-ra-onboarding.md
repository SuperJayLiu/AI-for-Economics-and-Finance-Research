# AI-Assisted RA Onboarding

## What Problem This Solves

Research assistants need clear rules before using AI on shared projects.

## Onboarding Topics

- project purpose
- folder structure
- data safety rules
- Git workflow
- AI-use policy
- code review
- paper reading standards
- documentation expectations
- research logs

## Minimum RA Rules

- do not upload restricted data
- do not edit raw data
- use branches for code changes
- log AI-assisted work
- ask before using AI on coauthored drafts
- verify outputs before reporting them

## Suggested Exercise

Give the RA a small public-data task. Ask them to use AI to propose a plan, run the code, inspect outputs, record the AI-use log, and explain what they verified.
