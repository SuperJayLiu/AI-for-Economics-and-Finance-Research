# Two Session Intro Workshop

## Session 1: Foundations

Topics:

- what LLMs are and are not
- hallucinations and fake citations
- responsible use and disclosure
- core AI skills for research
- AI skills management
- evaluating tools and features

Exercise:

- turn a vague research request into a structured AI workflow with verification steps

## Session 2: Applied Workflows

Topics:

- ChatGPT and Claude project setups
- empirical research and coding
- GitHub and version control
- data safety by data type
- research logs
- human-in-the-loop automation

Exercise:

- build a literature review or empirical paper project with project instructions, safe inputs, expected outputs, and a verification checklist
