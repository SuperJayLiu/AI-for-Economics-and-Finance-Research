# Reproducibility

AI-assisted empirical work should be reproducible by a human researcher.

Minimum practices:

- keep raw data separate from cleaned data
- document variable construction
- record package versions
- save code that produces every table and figure
- review AI-generated code before running it
- use version control for scripts and manuscripts
- keep an AI-assisted change log

Before relying on an AI-generated code edit, run a small example where you know the expected output.
