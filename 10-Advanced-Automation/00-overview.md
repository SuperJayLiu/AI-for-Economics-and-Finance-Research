# Advanced Automation

Advanced automation is useful only when verification is stronger than automation.

| Level | Name | Example |
| --- | --- | --- |
| 0 | Simple chat | Ask AI to explain a method |
| 1 | Guided workflow | Use AI to critique an introduction |
| 2 | AI project | Dedicated workspace for a paper |
| 3 | Repeatable workflow | Same steps for every paper or dataset |
| 4 | Semi-automation | Batch summarize papers or review code |
| 5 | Human-in-the-loop agent | AI searches, organizes, drafts, and flags issues, but human approves |

Do not automate tasks you cannot verify.
